# Egenskabsværdityper

## Beskrivelse

Forskellige typer af værdier, som en egenskab kan have i et givet system eller kontekst. For eksempel kan egenskabsværdier være numeriske, tekstbaserede, booleske (sand/falsk), eller andre typer af data.

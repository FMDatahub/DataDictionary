# Qto_BuildingBaseQuantities

BuildingBaseQuantities indeholder kvantitative oplysninger om forskellige bygningskomponenter og bygningens overordnede egenskaber.

- [Height](../Properties/Form/Height.md)
- EavesHeight
- FootPrintArea
- GrossFloorArea
- NetFloorArea
- GrossVolume
- NetVolume
